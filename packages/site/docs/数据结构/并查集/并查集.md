# 并查集(UnionFindSet)

也叫做不相交集合(Disjoint Set)

## 两个核心操作

- 查找(Find):查找元素所在的集合(指广义的数据集合)
- 合并(Union):将两个元素所在的集合合并为一个集合

## 两种常见速录

1. Quick Find
   查找(Find) 时间复杂度:O(1)
   合并(Union) 时间复杂度:O(n)
2. Quick Union
   查找(Find) 时间复杂度:O(logn)
   合并(Union) 时间复杂度:O(logn)
