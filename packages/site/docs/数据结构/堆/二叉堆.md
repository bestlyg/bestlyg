# 二叉堆(BinaryHeap)

使用完全二叉树实现堆结构，又称完全二叉堆，一般使用数组实现

- 对于根节点下标是 0 的元素
- 父节点为(i-1)>>1
- 左子节点为 2\*i+1
- 右子节点为 2\*i+2
