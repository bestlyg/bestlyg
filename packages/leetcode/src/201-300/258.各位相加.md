# 258. 各位相加

> 链接：[258. 各位相加](https://leetcode-cn.com/problems/add-digits/)  
> 难度：中等  
> 标签：数学、数论、模拟  
> 简介：给定一个非负整数 num，反复将各个位上的数字相加，直到结果为一位数。返回这个结果。

## 题解 1 - c++

- 编辑时间：2022.03.03
- 执行用时：0ms
- 内存消耗：5.7MB
- 编程语言：c++
- 解法介绍：递归遍历。

```cpp
class Solution {
   public:
    int addDigits(int num) {
        if (num < 10) return num;
        int ans = 0;
        for (; num; num /= 10) ans += num % 10;
        return addDigits(ans);
    }
};
```
