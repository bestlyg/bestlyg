# 912. 排序数组

> 链接：[912. 排序数组](https://leetcode-cn.com/problems/sort-an-array/)  
> 难度：中等  
> 标签：  
> 简介：给你一个整数数组 nums，请你将该数组升序排列。

## 题解 1 - typescript

- 编辑时间：2021.05.07
- 执行用时：156ms
- 内存消耗：51.9mb
- 编程语言：typescript
- 解法介绍：利用内部排序。

```typescript
function sortArray(nums: number[]): number[] {
  return nums.sort((a, b) => a - b);
}
```
