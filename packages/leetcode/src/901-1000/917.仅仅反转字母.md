# 917. 仅仅反转字母
    
> 链接：[917. 仅仅反转字母](https://leetcode-cn.com/problems/reverse-only-letters/)  
> 难度：简单  
> 标签：双指针、字符串  
> 简介：返回反转后的 s 。
      
## 题解 1 - c++
- 编辑时间：2022.02.23
- 执行用时：0ms
- 内存消耗：5.9MB
- 编程语言：c++
- 解法介绍：双指针。
```c++
class Solution {
   public:
    string reverseOnlyLetters(string s) {
        for (int l = 0, r = s.size() - 1; l < r; l++, r--) {
            while (l < s.size() && !isalpha(s[l])) l++;
            while (r >= 0 && !isalpha(s[r])) r--;
            if (l < r) swap(s[l], s[r]);
        }
        return s;
    }
};
```
