# 319. 灯泡开关
    
> 链接：[319. 灯泡开关](https://leetcode-cn.com/problems/bulb-switcher/)  
> 难度：中等  
> 标签：脑筋急转弯、数学  
> 简介：找出并返回 n 轮后有多少个亮着的灯泡。
      
## 题解 1 - typescript
- 编辑时间：2021.11.15
- 执行用时：76ms
- 内存消耗：39.4mb
- 编程语言：typescript
- 解法介绍：求出完全平方数即亮灯个数。
```typescript
function bulbSwitch(n: number): number {
          return Math.floor(Math.sqrt(n + 0.5));
        }
```
