# 507. 完美数
    
> 链接：[507. 完美数](https://leetcode-cn.com/problems/perfect-number/)  
> 难度：简单  
> 标签：数学  
> 简介：对于一个 正整数，如果它和除了它自身以外的所有 正因子 之和相等，我们称它为 「完美数」。
      
## 题解 1 - c++
- 编辑时间：2021.12.31
- 执行用时：0ms
- 内存消耗：5.9mb
- 编程语言：c++
- 解法介绍：遍历后获取每个因数。
```c++
class Solution {
   public:
    bool checkPerfectNumber(int num) {
        if (num == 1) return 0;
        int ans = 1;
        for (int i = 2; i * i <= num; i++) {
            if (num % i) continue;
            ans += i;
            if (i * i < num) ans += num / i;
        }
        return ans == num;
    }
};
```
