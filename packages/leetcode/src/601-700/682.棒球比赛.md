# 682. 棒球比赛

> 链接：[682. 棒球比赛](https://leetcode-cn.com/problems/baseball-game/)  
> 难度：简单  
> 标签：栈、数组、模拟  
> 简介：请你返回记录中所有得分的总和。

## 题解 1 - typescript

- 编辑时间：2021.03.19
- 执行用时：80ms
- 内存消耗：39.9MB
- 编程语言：typescript
- 解法介绍：dfs。

```typescript
class Solution {
   public:
    int calPoints(vector<string> &ops) {
        vector<int> list;
        for (auto &op : ops) {
            int n = list.size();
            if (op == "+") {
                list.push_back(list[n - 1] + list[n - 2]);
            } else if (op == "D") {
                list.push_back(list[n - 1] * 2);
            } else if (op == "C") {
                list.pop_back();
            } else {
                list.push_back(stoi(op));
            }
        }
        int ans = 0;
        for (auto &num : list) ans += num;
        return ans;
    }
};
```
