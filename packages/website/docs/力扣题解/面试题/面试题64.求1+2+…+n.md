# 面试题 64. 求 1+2+…+n

> 链接：[面试题 64. 求 1+2+…+n](https://leetcode-cn.com/problems/qiu-12n-lcof/)  
> 标签：  
> 简介：求 1+2+...+n ，要求不能使用乘除法、for、while、if、else、switch、case 等关键字及条件判断语句（A?B:C）。

## 题解 1 - typescript

- 编辑时间：2020.6.2
- 执行用时：56ms
- 内存消耗：32.2MB
- 编程语言：typescript
- 解法介绍：利用等差数列直接求和。

```typescript
var sumNums = function (n: number): number {
  return ((1 + n) * n) / 2;
};
```

## 题解 2 - typescript

- 编辑时间：2020.6.2
- 执行用时：80ms
- 内存消耗：34.7MB
- 编程语言：typescript
- 解法介绍：递归求和。

```typescript
var sumNums = function (n: number): number {
  n && (n += sumNums(n - 1));
  return n;
};
```

## 题解 3 - typescript

- 编辑时间：2020.6.2
- 执行用时：64ms
- 内存消耗：34.9MB
- 编程语言：typescript
- 解法介绍：递归求和。

```typescript
var sumNums = function (n: number): number {
  const c = (num: number, count: number): number => (count === 1 ? num : num + c(num, count - 1));
  return c(n, n + 1) >> 1;
};
```
