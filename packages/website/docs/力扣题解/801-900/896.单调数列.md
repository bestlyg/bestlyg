# 896. 单调数列

> 链接：[896. 单调数列](https://leetcode-cn.com/problems/monotonic-array/)  
> 难度：简单  
> 标签：数组  
> 简介：如果数组是单调递增或单调递减的，那么它是单调的。

## 题解 1 - typescript

- 编辑时间：2021.2.28
- 执行用时：104ms
- 内存消耗：44.1mb
- 编程语言：typescript
- 解法介绍：分别判断单调性。

```typescript
function isMonotonic(A: number[]): boolean {
  const len = A.length - 1;
  let f = true;
  for (let i = 0; i < len; i++) {
    if (A[i] > A[i + 1]) {
      f = false;
      break;
    }
  }
  if (f) return true;
  for (let i = 0; i < len; i++) {
    if (A[i] < A[i + 1]) {
      return false;
    }
  }
  return true;
}
```
