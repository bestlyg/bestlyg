# 10. 正则表达式匹配

> 链接：[10. 正则表达式匹配](https://leetcode-cn.com/problems/regular-expression-matching/)  
> 难度：困难  
> 标签：字符串、动态规划、回溯算法  
> 简介：给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 '.' 和 '\*' 的正则表达式匹配。

## 题解 1 - javascript

- 编辑时间：2020.4.12
- 执行用时：92ms
- 内存消耗：34.5MB
- 编程语言：javascript
- 解法介绍：直接偷懒用新建正则判断。

```javascript
/**
 * @param {string} s
 * @param {string} p
 * @return {boolean}
 */
var isMatch = (s, p) => new RegExp(`^${p}$`).test(s);
```
