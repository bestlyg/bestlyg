# 65. 有效数字

> 链接：[65. 有效数字](https://leetcode-cn.com/problems/valid-number/)  
> 难度：困难  
> 标签：数学、字符串  
> 简介：验证给定的字符串是否可以解释为十进制数字。

## 题解 1 - javascript

- 编辑时间：2020.4.7
- 执行用时：108ms
- 内存消耗：36.4MB
- 编程语言：javascript
- 解法介绍：使用 Number 方法直接可判断。

```javascript
/**
 * @param {string} s
 * @return {boolean}
 */
var isNumber = function (s) {
  if (s.trim() === '') return false;
  const result = Number(s);
  return !Number.isNaN(result);
};
```
