# 217. 存在重复元素

> 链接：[217. 存在重复元素](https://leetcode-cn.com/problems/contains-duplicate/)  
> 难度：简单  
> 标签：数组、哈希表  
> 简介：给定一个整数数组，判断是否存在重复元素。

## 题解 1 - typescript

- 编辑时间：2020.12.13
- 执行用时：100ms
- 内存消耗：44.5mb
- 编程语言：typescript
- 解法介绍：利用 set 去重。

```typescript
function containsDuplicate(nums: number[]): boolean {
  return new Set(nums).size !== nums.length;
}
```
