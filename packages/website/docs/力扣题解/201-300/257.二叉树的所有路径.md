# 257. 二叉树的所有路径

> 链接：[257. 二叉树的所有路径](https://leetcode-cn.com/problems/binary-tree-paths/)  
> 难度：简单  
> 标签：树、深度优先搜索  
> 简介：给定一个二叉树，返回所有从根节点到叶子节点的路径。

## 题解 1 - javascript

- 编辑时间：2020.9.4
- 执行用时：88ms
- 内存消耗：39.7mb
- 编程语言：javascript
- 解法介绍：函数式编程，深度优先搜索。

```javascript
/**
 * @param {TreeNode} root
 * @return {string[]}
 */
var binaryTreePaths = function (root) {
  if (!root) return [];
  else if (!root.left && !root.right) return [root.val + ''];
  else
    return ['left', 'right']
      .filter(sub => root[sub])
      .map(sub =>
        binaryTreePaths(root[sub])
          .filter(v => v)
          .map(v => `${root.val}->${v}`)
      )
      .flat();
};
```
