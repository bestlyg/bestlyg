# 485. 最大连续 1 的个数

> 链接：[485. 最大连续 1 的个数](https://leetcode-cn.com/problems/max-consecutive-ones/)  
> 难度：简单  
> 标签：数组  
> 简介：给定一个二进制数组， 计算其中最大连续 1 的个数。

## 题解 1 - typescript

- 编辑时间：2021.2.15
- 执行用时：96ms
- 内存消耗：41.4mb
- 编程语言：typescript
- 解法介绍：遍历。

```typescript
function findMaxConsecutiveOnes(nums: number[]): number {
  let max = 0;
  let num = 0;
  nums.forEach(v => {
    if (v & 1) {
      num++;
    } else {
      max = Math.max(max, num);
      num = 0;
    }
  });
  return Math.max(max, num);
}
```
