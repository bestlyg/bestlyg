# Trie

Trie 也叫做字典树、前缀树（Prefix Tree）、单词查找树，Trie 搜索字符串的效率主要跟字符串的长度有关。

## Trie 的优点

- 搜索前缀的效率主要跟前缀的长度有关

## Trie 的缺点

- 需要耗费大量的内存，
