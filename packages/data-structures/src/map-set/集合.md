# 集合(Set)

也叫做字典，键值对的匹配，两个元素的集之间元素相互“对应”的关系。

- 可以利用 Map 储存键，值只储存 null 实现。

## 常用映射

- HashMap
  - 底层使用哈希表
- ListMap
  - 底层使用链表
- TreeMap
  - 底层使用树形结构
