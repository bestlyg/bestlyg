# 1822. 数组元素积的符号
    
> 链接：[1822. 数组元素积的符号](https://leetcode.cn/problems/sign-of-the-product-of-an-array/)  
> 难度：简单  
> 标签：数组、数学  
> 简介：给你一个整数数组 nums 。令 product 为数组 nums 中所有元素值的乘积。
      
## 题解 1 - cpp
- 编辑时间：2022.10.27
- 执行用时：4ms
- 内存消耗：9.9MB
- 编程语言：cpp
- 解法介绍：遍历检查有几个负数。
```cpp
class Solution {
public:
    int arraySign(vector<int>& nums) {
        int cnt = 0;
        for (auto &num : nums) {
            if (num == 0) return 0;
            else if (num < 0) cnt ^= 1;
        }
        return cnt ? -1 : 1;
    }
};
```
