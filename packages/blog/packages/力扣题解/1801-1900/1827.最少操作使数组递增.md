# 1827. 最少操作使数组递增

> 链接：[1827. 最少操作使数组递增](https://leetcode.cn/problems/minimum-operations-to-make-the-array-increasing/)  
> 难度：简单  
> 标签：贪心、数组  
> 简介：请你返回使 nums 严格递增 的 最少 操作次数。

## 题解 1 - cpp

- 编辑时间：2022.12.11
- 执行用时：16ms
- 内存消耗：15.3MB
- 编程语言：cpp
- 解法介绍：遍历。

```cpp
class Solution {
public:
    int minOperations(vector<int>& nums) {
        int ans = 0;
        for (int i = 1; i < nums.size(); i++) {
            if (nums[i] <= nums[i - 1]) {
                ans += nums[i - 1] - nums[i] + 1;
                nums[i] = nums[i - 1] + 1;
            }
        }
        return ans;
    }
};
```
