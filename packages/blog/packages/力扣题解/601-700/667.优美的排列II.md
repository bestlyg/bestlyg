# 667. 优美的排列 II
    
> 链接：[667. 优美的排列 II](https://leetcode.cn/problems/beautiful-arrangement-ii/)  
> 难度：中等  
> 标签：数组、数学  
> 简介：给你两个整数 n 和 k ，请你构造一个答案列表 answer ，该列表应当包含从 1 到 n 的 n 个不同正整数。
      
## 题解 1 - cpp
- 编辑时间：2022.09.08
- 执行用时：28ms
- 内存消耗：8.9MB
- 编程语言：cpp
