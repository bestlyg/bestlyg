# 817. 链表组件

> 链接：[817. 链表组件](https://leetcode.cn/problems/linked-list-components/)  
> 难度：中等  
> 标签：数组、哈希表、链表  
> 简介：返回列表 nums 中组件的个数。

## 题解 1 - cpp

- 编辑时间：2022.10.12
- 执行用时：24ms
- 内存消耗：20.8MB
- 编程语言：cpp
- 解法介绍：遍历。

```cpp
class Solution {
public:
    int numComponents(ListNode* head, vector<int>& nums) {
        unordered_set<int> s(nums.begin(), nums.end());
        ListNode *p = head;
        while (p && !s.count(p->val)) p = p->next;
        int ans = 0;
        while (p) {
            while (p && s.count(p->val)) p = p->next;
            ans++;
            while (p && !s.count(p->val)) p = p->next;
        }
        return ans;
    }
};
```
