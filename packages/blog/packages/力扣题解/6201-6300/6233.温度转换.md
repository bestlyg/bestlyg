# 6233. 温度转换

> 链接：[6233. 温度转换](https://leetcode.cn/problems/convert-the-temperature/)  
> 难度：简单  
> 标签：  
> 简介：返回数组 ans 。与实际答案误差不超过 10-5 的会视为正确答案。

## 题解 1 - cpp

- 编辑时间：2022.11.13
- 执行用时：0ms
- 内存消耗：5.9MB
- 编程语言：cpp
- 解法介绍：直接计算。

```cpp
class Solution {
public:
    vector<double> convertTemperature(double celsius) {
        vector<double> ans(2);
        ans[0] = celsius + 273.15;
        ans[1] = celsius * 1.80 + 32.00;
        return ans;
    }
};
```
