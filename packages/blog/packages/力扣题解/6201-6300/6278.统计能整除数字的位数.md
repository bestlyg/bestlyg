# 6278. 统计能整除数字的位数
    
> 链接：[6278. 统计能整除数字的位数](https://leetcode.cn/problems/count-the-digits-that-divide-a-number/)  
> 难度：简单  
> 标签：  
> 简介：给你一个整数 num ，返回 num 中能整除 num 的数位的数目。
      
## 题解 1 - cpp
- 编辑时间：2023.01.01
- 执行用时：0ms
- 内存消耗：5.8MB
- 编程语言：cpp
- 解法介绍：遍历。
```cpp
class Solution {
public:
    int countDigits(int num) {
        int ans = 0;
        for (int v = num; v; v /= 10) {
            int sub = v % 10;
            if (num % sub == 0) ans++;
        }
        return ans;
    }
};
```

## 题解 2 - rust
- 编辑时间：2023.01.01
- 执行用时：0ms
- 内存消耗：2.1MB
- 编程语言：rust
- 解法介绍：遍历。
```rust
impl Solution {
    pub fn count_digits(num: i32) -> i32 {
        let mut ans = 0;
        let mut tmp = num;
        while tmp != 0 {
            let v = tmp % 10;
            if num % v == 0 {
                ans += 1;
            }
            tmp /= 10;
        }
        ans
    }
}
```
