# 6271. 礼盒的最大甜蜜度
    
> 链接：[6271. 礼盒的最大甜蜜度](https://leetcode.cn/problems/maximum-tastiness-of-candy-basket/)  
> 难度：中等  
> 标签：  
> 简介：返回礼盒的 最大 甜蜜度。
      
## 题解 1 - cpp
- 编辑时间：2022.12.25
- 执行用时：172ms
- 内存消耗：49.4MB
- 编程语言：cpp
- 解法介绍：二分答案。
```cpp
class Solution {
public:
    int n, k;
    vector<int> price;
    int maximumTastiness(vector<int>& price, int k) {
        sort(price.begin(), price.end());
        n = price.size();
        this->k = k;
        this->price = price;
        int l = 0, r = price[n - 1] - price[0], m;
        while (l < r) {
            m = (l + r + 1) / 2;
            if (check(m)) l = m;
            else r = m - 1;
        }
        return l;
    }
    bool check(int num) {
        int cnt = 1, cur = price[0];
        for (int i = 1; i < n; i++) {
            if (cur + num <= price[i]) {
                cnt++;
                cur = price[i];
            }
        }
        return cnt >= k;
    }
};
```
