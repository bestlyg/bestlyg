# 6220. 可被三整除的偶数的平均值

> 链接：[6220. 可被三整除的偶数的平均值](https://leetcode.cn/problems/most-popular-video-creator/)  
> 难度：简单  
> 标签：  
> 简介：给你一个由正整数组成的整数数组 nums ，返回其中可被 3 整除的所有偶数的平均值。

## 题解 1 - cpp

- 编辑时间：2022.10.30
- 执行用时：8ms
- 内存消耗：13.3MB
- 编程语言：cpp
- 解法介绍：遍历。

```cpp
class Solution {
public:
    int averageValue(vector<int>& nums) {
        int sum = 0, cnt = 0;
        for (auto &num : nums) {
            if (num % 3 == 0 && num % 2 == 0) {
                sum += num;
                cnt += 1;
            }
        }
        return cnt == 0 ? 0 : sum / cnt;
    }
};
```
